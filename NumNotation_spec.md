# NumNotation 规范 v0.2

NumNotation 是一种基于简谱数字音级的纯文本记谱格式，
时值写法完全遵循简谱传统（增时线 `-`、减时线 `_`、附点 `.`），
专为古典吉他谱、钢琴谱设计，可转换为 LilyPond 代码。

---

## 一、文件头

```
%numnotation-0.2

X:1
T:练习曲
C:Fernando Sor
M:3/4
Q:1/4=96
K:1=C
I:instrument=guitar
```

| 字段 | 说明 |
|------|------|
| `X:` | 曲目编号（必须） |
| `T:` | 标题 |
| `C:` | 作曲者 |
| `M:` | 拍号，如 `4/4` `3/4` `6/8` |
| `Q:` | 速度，如 `1/4=120` 或 `"Andante" 1/4=72` |
| `K:` | 调号 |
| `I:` | 乐器及其他指令 |

`K:` 或 `I:instrument=` 出现后，之后的内容为乐谱正文。

---

## 二、调号

```
K:1=C      % C大调
K:1=G      % G大调
K:1=Bb     % 降B大调
K:6=A      % A小调（自动识别为 minor）
K:1=C m    % C小调（显式指定）
```

## 三、时值

### 3.1 核心规则（与简谱完全一致）

**基本音符（裸数字，不加任何修饰）= 四分音符（一拍）**

### 3.2 增时线（延长）

`-` 是独立 token，写在音符后、空格分隔，每个 `-` 延长一拍：

```
1             % 四分音符   1拍
1 -           % 二分音符   2拍
1 - -         % 附点二分   3拍（等同于 1-.）
1 - - -       % 全音符     4拍
```

### 3.3 减时线（缩短）

`_` 紧跟数字后（无空格），每个 `_` 缩短一半：

```
1_            % 八分音符   1/2拍
1__           % 十六分音符 1/4拍
1___          % 三十二分音符 1/8拍
```

### 3.4 附点

`.` 紧跟在音符（或减时线）后，延长原时值的一半：

```
1.            % 附点四分   1.5拍
1_.           % 附点八分   3/4拍
1__.          % 附点十六   3/8拍
1..           % 双附点四分 1.75拍
```

附点二分音符有两种等价写法：

```
1 - -         % 推荐（视觉更直观）
1 -.          % 亦可（减少一个 token）
```

### 3.5 时值速查表

```
写法          时值名称        拍数
-------------------------------------------
1 - - -       全音符          4拍
1 -           二分音符        2拍
1             四分音符        1拍
1_            八分音符        1/2拍
1__           十六分音符      1/4拍
1___          三十二分音符    1/8拍

1 - -         附点二分        3拍
1.            附点四分        1.5拍
1_.           附点八分        3/4拍
```

## 四、音高

### 4.1 音级

```
1 2 3 4 5 6 7    % do re mi fa sol la si
```

### 4.2 八度

八度修饰符紧跟数字，位于减时线之前：

```
1,   2,   3,     % 低一个八度
1,,  2,,         % 低两个八度
1'   2'   3'     % 高一个八度
1''  2''         % 高两个八度
```

示例：`1'_`（高八度八分音符）、`5,__`（低八度十六分音符）

### 4.3 升降号（前置）

```
#1   % 升1
b3   % 降3
=2   % 还原2
```

## 五、休止符

```
0             % 四分休止（一拍）
0 -           % 二分休止
0 - - -       % 全音符休止
0_            % 八分休止
0__           % 十六分休止
0.            % 附点四分休止
```

## 六、和弦

`[ ]` 包裹多个音符，时值标注在括号外的第一个音符位置上（即时值跟在 `]` 后紧接或以 `-` 延长）：

```
[1 3 5]           % 四分和弦
[1 3 5] -         % 二分和弦
[1, 5, 3]_        % 八分和弦（减时线紧跟 ]）
[1, 5, 3]. -      % 附点四分后再延长…（特殊情况）
```

---

## 七、小节线与重复

```
|         % 小节线
||        % 段落双小节线
|]        % 终止线
|:        % 重复开始
:|        % 重复结束

|: 1 2 3 4 [1 5 - :| [2 1 - - - |   % 第一、二房
```

## 八、多声部

```
V:1 stem=up
V:2 stem=down

V:1
5_ 3_ 2_ | 1 - - |

V:2
[1, 5,] 0 0 | [1, 5,] - - |
```

钢琴高低谱表：

```
V:1 clef=treble stem=up
V:2 clef=bass stem=down
```

## 十二、古典吉他专用

```
1[p]          % 右手拇指
2_[i]         % 右手食指（八分音符）
3[m]          % 右手中指
4[a]          % 右手无名指

1{1}          % 左手食指
2{2}          % 左手中指
3{3}          % 左手无名指
4{4}          % 左手小指

1(s2)         % 第2弦
P:V           % 第五把位
P:CVII        % 大横按七把位
P:BIII        % 小横按三把位

!harm!        % 泛音
!pizz!        % 止音
```

---

## 十三、钢琴专用

```
1<3>          % 钢琴指法（第3指）
!ped!         % 踏板踩下
!pup!         % 踏板抬起
```

## 十五、完整示例

### 古典吉他谱（C大调 3/4）

```
%numnotation-0.2
X:1
T:Etude in C
C:Carcassi
M:3/4
Q:1/4=96
K:1=C
I:instrument=guitar

V:1 stem=up
V:2 stem=down

V:1
!mf! (3_ 2_ 1_) 0. | 2 - - | (5_ 4_ 3_) 0. | 5 - - |

V:2
[1, 5, 3] 0 0 | [1, 5, 3] 0 0 | [4, 1 6,] 0 0 | [5, 2 5,] 0 0 |
```

### 钢琴谱（G大调 4/4）

```
%numnotation-0.2
X:1
T:Sonatina in G
M:4/4
Q:"Allegretto" 1/4=120
K:1=G
I:instrument=piano

V:1 clef=treble stem=up
V:2 clef=bass stem=down

V:1
!mf! 1_ 2_ 3_ 4_ 5_ 6_ 7_ 1'_ | 5 - 5 - | 6_ 5_ 4_ 3_ 2_ 1_ 7_ 6_ | 2 - - - |

V:2
[1,, 5,, 3,] - [1,, 5,, 3,] - | [1,, 5,, 3,] - [1,, 5,, 3,] - |
[4,, 1, 2,] - [4,, 1, 2,] - | [5,, 2, 7,,] - - - |
```

## 十六、解析规则

一个完整音符 token 的结构：

```
[升降号] 音级 [八度修饰] [减时线] [附点] [右手指法] [左手指法] [弦号] [钢琴指法]
 #/b/=   1-7    '/,        _*       .*      [pima]     {1-4}    (sN)    <N>
```

`-` 是独立 token（增时线），出现在音符 token 之后，空格分隔。
一行内可以混写多个声部或小节，`|` 作为小节分隔。
